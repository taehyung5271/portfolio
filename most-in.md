# Most-In

> 직원관리시스템(출퇴근/스케줄/공지/발주) 관리 기능을 한 곳에

## 한 줄 소개
대형 리테일 매장에서 사용할 수 있는 **직원 관리 시스템**으로, 출퇴근/스케줄/공지/발주 등 운영에 필요한 기능을 통합한 프로젝트입니다.

## 프로젝트 개요
- **목표**: 매장 운영 데이터를 한 곳에서 관리하여 업무 효율 향상
- **구성**: 클라이언트 단(Single Device) 중심 + 로컬 DB 기반

## 주요 기능
- 직원 정보 관리(직급/권한 분리)
- 출퇴근 등록 및 조회
- 스케줄 관리
- 공지사항 게시판
- 발주/주문 관리

## 내가 맡은 역할
- DB 모델링(테이블/관계 설계)
- 화면(리스트/상세) UI 구성
- 기능 로직 구현 및 리팩토링

## 기술 스택
- Android: Java / Kotlin(혼용 또는 전환 중)
- Local DB: SQLite
- Architecture: (추후 입력: MVC/MVVM 등)

## 트러블슈팅 / 개선 포인트
- 데이터 정규화(예: goods 테이블 분리, 참조 구조 도입)
- 출퇴근 등록 UI/로직 리팩토링 진행
- (예정) 로그인 테이블 추가 및 권한별 화면 제어 강화

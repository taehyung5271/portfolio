# NoteCam

> AI 영상 노트 필기 도우미

## 한 줄 소개
영상 시청 중 중요한 장면을 캡처하고 OCR로 텍스트를 추출하여, **노트를 자동으로 저장/조회**할 수 있는 Android 앱 프로젝트입니다.

## 프로젝트 개요
- **목표**: “영상에서 얻는 지식”을 빠르게 기록하고 다시 꺼내보기
- **핵심 흐름**: 캡처 → OCR → 저장 → 조회/상세 → 편집/삭제

## 구현한 기능(현재)
- OCR 결과 저장/조회 기능 구현 완료
- 로컬 데이터 기반으로 노트 관리

## 개발 예정 기능
- 상세보기 UI 고도화
- 노트 수정/삭제
- (보류) 백업(Firebase 등) 연동

## 기술 스택
- Android: Kotlin
- Local Storage: (예: Room/SQLite 중 사용 중인 것으로 수정)
- OCR: (사용 라이브러리/엔진 기입)

## 트러블슈팅 / 개선 포인트
- OCR 결과의 정확도와 후처리(공백/줄바꿈/특수문자 처리)
- 노트 검색/필터 기능 추가 고려
- 백업 기능(클라우드 동기화) 여부 재검토
